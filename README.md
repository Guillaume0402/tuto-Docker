## 🚀 Installation rapide

### Prérequis

-   Docker Desktop 4.0+ ([Installation](https://docs.docker.com/desktop/))
-   Git ([Installation](https://git-scm.com/downloads))
-   4GB RAM minimum, 8GB recommandé

### Démarrage en 3 étapes

```bash
# 1. Cloner le projet
git clone https://github.com/votre-repo/tuto-docker.git
cd tuto-docker

# 2. Configuration de l'environnement
cp .env.example .env
# Optionnel: éditer .env selon vos besoins

# 3. Lancement des conteneurs
docker-compose up -d
```

### 🌐 Accès aux services

| Service        | URL                   | Utilisateur | Mot de passe    |
| -------------- | --------------------- | ----------- | --------------- |
| **Site web**   | http://localhost:8080 | -           | -               |
| **phpMyAdmin** | http://localhost:8081 | `root`      | `root_password` |
| **Mailhog**    | http://localhost:8025 | -           | -               |

## 🔧 Commandes de développement

```bash
# Gestion des conteneurs
make up              # Démarrer tous les services
make down            # Arrêter tous les services
make restart         # Redémarrer les services
make clean           # Nettoyer les données (attention!)

# Logs et monitoring
make logs            # Voir tous les logs
make logs-php        # Logs du conteneur PHP uniquement
make logs-mysql      # Logs du conteneur MySQL

# Base de données
make db-reset        # Réinitialiser la base de données
make db-backup       # Sauvegarder la base
make db-restore      # Restaurer une sauvegarde

# Développement
make shell-php       # Accès shell du conteneur PHP
make shell-mysql     # Accès shell MySQL
make assets-watch    # Compilation SCSS en temps réel
```

## 🔐 Configuration avancée

### Variables d'environnement (.env)

```env
# Application
APP_NAME="Formation Docker"
APP_URL="http://localhost:8080"
APP_ENV="development"

# Base de données
DB_HOST="mysql"
DB_PORT="3306"
DB_NAME="tuto_docker"
DB_USERNAME="root"
DB_PASSWORD="root_password"

# Email (Mailhog)
MAIL_HOST="mailhog"
MAIL_PORT="1025"
MAIL_FROM="no-reply@tuto-docker.local"

# Sécurité
SESSION_LIFETIME="7200"
CSRF_TOKEN_NAME="_token"
```
